# 통계작성의 중지승인(협의) 고시(화훼재배현황)

1. 통계의 명칭 : 화훼재배현황
　
2. 통계작성기관의 명칭 : 농림축산식품부
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제114032호 [1998.10.13.]
　
4. 통계작성의 중지 연월일 : 2026.08.31.
　
5. 통계작성의 중지 사유 : 신규 통계(조사통계)로 전환하기 위한 작성 중지
