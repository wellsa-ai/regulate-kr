# 통계작성의 작성승인(협의) 고시(태안군청년통계)

1. 통계의 명칭 : 태안군청년통계
　
2. 통계작성기관의 명칭 : 충청남도 태안군
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제704004호 [2025.12.12.]
　
4. 통계작성의 목적 : 태안군 청년의 인구, 주거, 일자리, 창업, 복지, 건강 등에 대한 통계를 생산, 분석하여 청년 정책 수립에 활용
　
5. 통계작성의 대상
　    - 대상 : 개인/가구
　    - 모집단 : 해당없음
　    - 대상 범위 및 규모 : 작성기준 시점 태안군에 거주하는 청년(태안군 청년 기본 조례 : 18세~45세)
　    - 대상지역 : 시군구
　
6. 통계작성의 주기 : 2년
　
7. 통계작성의 방법 : 가공통계/일반통계
