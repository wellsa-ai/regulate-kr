# 금강Ⅱ지구 대단위농업개발사업 지형도면 고시

1. 명칭 : 금강Ⅱ지구 대단위농업개발사업
　
 2. 위치(면적) : 1도 1시 19개 읍·면·동 일원 (1,720필지 4,482,636㎡)

 3. 지구 지정일 : 1989년 12월 2일(농림수산부고시 제89-56호)
　
 4. 지형도면 고시일 : 2013년 8월  일
　
 5. 지형도면 : 도면 게재 생략
