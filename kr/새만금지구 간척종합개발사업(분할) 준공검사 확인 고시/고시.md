# 새만금지구 간척종합개발사업(분할) 준공검사 확인 고시

1. 면허일자 : 1991년 10월 17일(농림수산부고시 제91-32호 : 특개 제3호)
　
 2. 사 업 명 : 새만금지구 간척종합개발사업(외곽시설)
　
 3. 준공검사 확인을 받은 자의 성명·주소·전화번호
   가. 성  명 : 한국농어촌공사사장
   나. 주  소 : 전라남도 나주시 그린로 20
   다. 연락처 : 061-338-5368
　
 4. 매립목적 : 농지조성과 용수개발을 주목적으로 하는 간척종합개발
　
 5. 공사위치 : 전라북도 군산시, 김제시, 부안군 지선의 새만금 간척종합개발사업 지역의 공유수면
　
 6. 준공면적 : 총 7,071,257.3㎡(외곽시설_가력선착장 제외)
   가. 국가귀속 : 2,723,233.4㎡
                 (1,2호 방조제 : 1,336,095.9㎡, 3,4호 방조제 : 1,387,137.5㎡)
   나. 한국농어촌공사(기금수탁관리자) : 4,348,023.9㎡
                   (1,2호 방조제 : 1,243,518.6㎡, 3,4호 방조제 : 3,104,505.3㎡)
　
 7. 매립공사의 시행기간 : 1991년 11월 13일 ～2011년 12월 31일
　
 8. 총사업비 : 2,942,026,821천원
               (1,2호 방조제 : 1,941,744,002천원, 3,4호 방조제 : 1,000,282,819천원)
　
 9. 기타사항 : 기타 문의사항은 한국농어촌공사(061-338-5368)으로 문의하시기 바랍니다.
