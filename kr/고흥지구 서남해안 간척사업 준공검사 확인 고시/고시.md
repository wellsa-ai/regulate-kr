# 고흥지구 서남해안 간척사업 준공검사 확인 고시

1. 면허일자 : 1991년 08월 23일(농림수산부 고시-91-25호)
　
 2. 사 업 명 : 고흥지구 서남해안 간척사업
　
 3. 준공검사 확인을 받은 자의 성명·주소·전화번호
   가. 성  명 : 고흥군수
   나. 주  소 : 전라남도 고흥군 고흥읍 옥하리 200번지
   다. 연락처 : 061-830-5471
　
 4. 매립목적 : 농지조성
　
 5. 공사위치 : 전라남도 고흥군 고흥읍, 풍양면, 도덕면, 두원면
　
 6. 준공면적 : 총 30,831,661.2㎡
   가. 국가귀속 : 13,320,875.3㎡(공공용지 13,320,875.3㎡)
   나. 한국농어촌공사(기금수탁관리자)관리 : 17,510,785.9㎡
　
 7. 매립공사의 시행기간 : 1991년 10월 31일 ～2008년 12월 30일
　
 8. 총사업비 : 396,913,211천원
　
 9. 기타사항 : 기타 문의사항은 고흥군(우.548-804 전남 고흥군 고흥읍 후동길10 고흥우주항공사업소, 전화번호: 061-830-5471)으로 문의하시기 바랍니다.
