# 통계작성의 승인(협의) 고시(음성군주거실태조사)

1. 통계의 명칭 : 음성군주거실태조사
　
2. 통계작성기관의 명칭 : 충청북도 음성군
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제680003호 [2025.05.13.]
　
4. 통계작성의 목적 : 음성군 주택수요ㆍ공급 추계를 위한 일반가구 주거실태 및 주택수요 파악
　
5. 통계작성의 대상
       - 대상 : 가구/가구
       - 모집단 : 음성군에 거주하는 일반가구
       - 대상 범위 및 규모 : 1,000가구
       - 대상지역 : 시군구
　
6. 통계작성의 주기 : 5년
　
7. 통계작성의 방법 : 조사통계/일반통계
