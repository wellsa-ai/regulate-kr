# 통계작성의 승인(협의) 고시(충청남도도민인권의식실태조사)

1. 통계의 명칭 : 충청남도도민인권의식실태조사
　
2. 통계작성기관의 명칭 : 충청남도
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제213014호 [2023.10.10.]
　
4. 통계작성의 목적 : 충청남도 인권상황을 체계적으로 진단ㆍ평가하여 충청남도 인권정책 추진에 필요한 기초자료로 활용하기 위해 통계 구축
　
5. 통계작성의 대상
       - 대상 : 개인/가구
       - 모집단 : 기숙시설, 특수사회시설 제외한 일반가구의 만 13세 이상 가구원
       - 대상 범위 및 규모 : 
         ㆍ 조사대상: 충청남도 거주 만13세 이상 도민(기숙시설, 특수사회시설 제외한 일반가구)
         ㆍ 조사규모: 충청남도 1,002가구 내외(105개 조사구)
       - 대상지역 : 시도
　
6. 통계작성의 주기 : 5년
　
7. 통계작성의 방법 : 조사통계/일반통계
