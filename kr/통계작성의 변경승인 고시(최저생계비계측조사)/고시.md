# 통계작성의 변경승인 고시(최저생계비계측조사)

1. 통계의 명칭 : 최저생계비계측조사
　
2. 통계작성기관의 명칭 : 보건복지부
　
3. 통계작성승인번호(작성승인일) : 제33104호(2004. 04. 16.)
　
4. 통계작성의 중지 연월일 : 2015. 05. 08.
　
5. 통계작성의 중지 사유 : 「국민기초생활 보장법」개정에 따라 최저생계비 결정을 위한 별도의 계측조사 근거가 삭제되어, 본 통계를 작성 중지
