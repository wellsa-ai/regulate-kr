# 통계작성의 변경승인고시(월세가격동향조사)

1. 통계의 명칭 : 월세가격동향조사
　
2. 통계작성기관의 명칭 : 한국감정원
　
3. 통계작성승인번호(작성승인일) : 제11671호(2009. 06. 16.)
　
4. 통계작성의 중지 연월일 : 2015. 07. 13.
　
5. 통계작성의 중지 사유 
  - 전세의 월세 전환 등 임대차 시장 내 전월세간 이동이 활발한 가운데 전세와 월세를 구분하여 조사하는 현재 조사체계는 임대차시장의 종합적 모니터링을 하는데 한계가 있는 바,
  - 기존의 월세가격동향조사 공표를 중지하는 대신 매매, 전세가격을 조사하는 전국주택가격동향조사와 월세가격을 조사하는 월세가격동향조사를 전국주택가격동향조사로 통합하여 조사지역과 표본수를 확대하는 통계개선을 실시하여 효율성과 활용성을 증진시키고자 함.
