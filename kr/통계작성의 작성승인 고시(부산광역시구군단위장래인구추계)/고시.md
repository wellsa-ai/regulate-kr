# 통계작성의 작성승인 고시(부산광역시구군단위장래인구추계)

1. 통계의 명칭 : 부산광역시구군단위장래인구추계
　
2. 통계작성기관의 명칭 : 부산광역시
　
3. 통계작성승인번호(작성승인일) : 제20214호(2015. 12. 18.)
　
4. 통계작성의 목적 : 인구변동의 실적자료를 이용하여 자치구별 과거인구를 확정하고, 장래인구를 추계하여 중장기 지역정책 기초자료 제공
　
5. 통계작성의 대상 : 2014~2033년 부산광역시 16개 구군별 상주인구(외국인 포함)
　
6. 통계작성의 주기  : 5년
　
7. 통계작성의 방법  : 가공통계
