# 통계작성의 변경승인(협의) 고시(경기도남양주시기본통계_2차)

1. 통계의 명칭: 경기도남양주시기본통계(2차)
　
2. 통계작성기관의 명칭: 경기도 남양주시
　
3. 통계작성승인(협의)번호 [작성승인(협의)일]: 제623001호 [2007.06.15.]
　
4. 통계작성의 변경 연월일: 2025.04.24.
　
5. 통계작성의 변경 내용과 사유: 세부내역 붙임참조
