# 통계작성의 승인(협의) 고시(경상북도노인등록통계)

1. 통계의 명칭 : 경상북도노인등록통계
　
2. 통계작성기관의 명칭 : 경상북도
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제216014호 [2023.11.29.]
　
4. 통계작성의 목적 : 경상북도 노인 현황을 파악하여 삶의 질 향상을 위한 노인복지정책수립의 기초자료 제공
　
5. 통계작성의 대상
       - 대상 : 개인
       - 모집단 : 작성기준년 12월 31일 현재 경상북도 거주 65세 이상 노인(55~64세 예비노인포함)
       - 대상 범위 및 규모 : 경상북도 거주 노인(65세 이상), 55~64세 예비노인 포함
       - 대상지역 : 시도
　
6. 통계작성의 주기 : 3년
　
7. 통계작성의 방법 : 가공통계/일반통계
