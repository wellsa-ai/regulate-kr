# 홍보지구 대단위농업개발사업 공유수면매립공사 준공검사 확인

1. 면허일자 : 1991년 11월 8일 (특개 27262-1281) (특개 제4호)
　
 2. 사 업 명 : 홍보지구 대단위농업개발사업
　
 3. 준공검사 확인을 받은 자의 성명·주소·연락처
    가. 성  명 : 한국농어촌공사 사장  김인식
    나. 주  소 : 전라남도 나주시 그린로 20
    다. 연락처 : 061-338-5387
　
 4. 매립목적 : 수자원확보 및 농어촌종합개발사업
　
 5. 공사위치 : 충남 홍성군 광천읍, 서부면, 은하면, 결성면, 보령시 오천면, 청소면, 천북면(1도 2시·군) 일원 공유수면
　
 6. 준공면적 : 총 15,856,474.8㎡
    가. 국가귀속(공공용지) : 15,856,474.8㎡
      (홍성호 : 4,715,766.1㎡, 보령호 : 11,140,708.7㎡)
　
 7. 매립공사의 시행기간 : 1991년 11월 8일 ~ 2016년 12월 30일
　
 8. 총사업비 : 46,831,314천원
      (홍성호 : 14,737,542천원, 보령호 : 32,093,772천원)
　
 9. 기타사항 : 기타 문의사항은 한국농어촌공사(061-338-5387)로 문의하시기 바랍니다.
