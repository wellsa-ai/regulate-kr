# 「합천 옥전 고분군」의 사적(문화재구역) 추가 지정

1. 고 시 명 : 「합천 옥전 고분군」의 사적(문화재구역) 추가 지정
 가. 대상 문화재
  1) 종    별 : 사적 제326호 합천 옥전 고분군 
  2) 소 재 지 : 경상남도 합천군 쌍책면 성산리 28번지 등
  3) 추가 지정면적(지번별 면적조서 붙임) : 30필지 112,852㎡
    ㅇ 당초 면적 19필지 131,064㎡, 추가 면적 30필지 112,852㎡, 합계 면적 49필지 243,916㎡
  4) 추가 지정사유 : 지표조사 결과 합천 옥전 고분군의 유구 분포 범위가 확대되고 있어, 합천 옥전 고분군의 문화재 가치 보존, 역사문화환경 보호, 주변지역 정비 등을 위해 문화재구역(지정구역)을 확대하여 보존·관리할 필요성이 있음
 나. 관리단체 : 경상남도 합천군(합천군수)
　
2. 추가 지정일 : 관보 고시일
　
3. 특기사항 
 ㅇ 「토지이용계획 기본법」제8조(지역·지구등의 지정 등)에 따른 지형도면 등은 ‘문화재공간정보(GIS)서비스(http://www.gis-heritage.go.kr) 알림마당-지형도면고시’에서 열람이 가능하고, ‘경상남도 가야문화유산과’ 및 ‘합천군 문화예술과’, ‘문화재청 보존정책과’에 관계도서를 비치하여 이해관계인에게 보이고 있습니다.
