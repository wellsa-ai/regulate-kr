# 통계작성의 승인(협의) 고시(국가기술자격수험자기초현황)

1. 통계의 명칭 : 국가기술자격수험자기초현황
　
2. 통계작성기관의 명칭 : 한국산업인력공단
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제387005호 [2026.04.07.]
　
4. 통계작성의 목적 : 실 응시자 기반 통계분석을 통해 국가인력수급정책 수립에 기여하고, 각 종목별 분석을 토대로 수험자의 경향을 분석하여 국가기술자격품질 관리에 활용
　
5. 통계작성의 대상
       - 대상 : 개인/개인
       - 모집단 : 국가기술자격 시험 응시자
       - 대상 범위 및 규모 : 한국산업인력공단 시행 국가기술자격 필기시험(전체) 및 실기시험(17 종목) 응시자[약 170만명]
       - 대상지역 : 전국
　
6. 통계작성의 주기 : 1년
　
7. 통계작성의 방법 : 보고통계/일반통계
