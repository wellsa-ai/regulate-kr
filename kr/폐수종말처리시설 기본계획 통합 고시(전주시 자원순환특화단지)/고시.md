# 폐수종말처리시설 기본계획 통합 고시(전주시 자원순환특화단지)

가. 폐수종말처리시설에서 처리하고자 하는 대상지역에 관한 사항
  ○ 전주시 자원순환특화단지[전북 전주시 완산구 정여립로 651(상림동 산188번지) 일원]
  ○ 면적 : (당초) 80,592㎡ → (변경) 80,009.3㎡
　
 나. 오염원 분포 및 폐수배출량과 그 예측에 관한 사항
  ○ 오염원 분포 : 전주시 자원순환특화단지에서 발생하는 오·폐수
  ○ 계획 오·폐수 발생량 : (당초) 224.8㎥/일→ (변경) 220.1㎥/일
　
 다. 처리계통도·처리능력 및 처리방법에 관한 사항
  ○ 처리계통도
    - 전처리 → 응집침전조 → 생물반응조(HANT) → 여과/소독 → 방류
  ○ 처리능력 : 230㎥/일
  ○ 처리방법 : 생물학적 고도처리공법(HANT)
　
 라. 처리된 폐수가 방류수역의 수질에 미치는 영향평가
  ○ 방류수역 : (당초) 저수지(안심제 → 기지제) → 전주천 → 만경강
            ⇒ (변경) 방류관 → 기존 우수관 → 중복천 → 삼천 → 만경강 
  ○ 방류수의 배출부하량
(단위 : mg/L)

 마. 폐수종말처리시설 설치·운영자에 관한 사항

 바. 부담금의 비용부담에 관한 사항
  ○ 시설설치비 : (당초) 3,905백만원 → (변경) 2,771백만원

 사. 토지 등의 수용 및 사용에 관한 사항 : 해당없음
