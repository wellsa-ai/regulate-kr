# 통계작성의 승인(협의) 고시(인구동태패널통계)

1. 통계의 명칭 : 인구동태패널통계
　
2. 통계작성기관의 명칭 : 통계청
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제101094호 [2025.07.16.]
　
4. 통계작성의 목적 : 특정 인구집단을 대상으로 개인의 경제ㆍ사회적 특성에 따른 혼인ㆍ출산 선택요인 및 이행ㆍ유지 비율에 대한 종단면 분석
　
5. 통계작성의 대상
       - 대상 : 개인/개인
       - 모집단 : 국내에서 출생 신고한 '83~'95년생 내국인
       - 대상 범위 및 규모 : 국내에서 출생 신고한 ’83~’95년생 내국인 약 860만명
* 주민등록번호가 없는 경우와 국외 출생자, 귀화자, 외국인, 사망자 제외
       - 대상지역 : 전국
　
6. 통계작성의 주기 : 1년
　
7. 통계작성의 방법 : 가공통계/일반통계
