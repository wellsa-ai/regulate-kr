# 통계작성의 승인(협의) 고시(중견기업수출동향)

1. 통계의 명칭 : 중견기업수출동향
　
2. 통계작성기관의 명칭 : 산업통상자원부
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제115037호 [2023.7.10.]
　
4. 통계작성의 목적 : 중견기업의 수출동향을 매분기 파악, 분석하여 중견기업 수출지원정책 수립에 필요한 기초자료로 활용
　
5. 통계작성의 대상 :  중견기업법 제2조에 근거한 중견기업 중 수출실적이 있는 기업 
　
6. 통계작성의 주기 : 분기
　
7. 통계작성의 방법 : 가공통계/일반통계
