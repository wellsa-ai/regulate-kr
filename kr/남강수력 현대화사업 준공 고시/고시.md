# 남강수력 현대화사업 준공 고시

1. 사업의 명칭 : 남강수력 현대화사업
　
  2. 사업시행자의 명칭 및 주소
    가. 명  칭 : 한국수자원공사
    나. 주  소 : 대전광역시 대덕구 신탄진로 200 (연축동 산6-2)
　
  3. 사업의 목적 
    장기간 사용(45년)으로 노후화된 남강수력 발전설비에 대하여 사고예방 및 안전성 확보, 기후 변화 등에 대응해 성능을 개선하고자 함
　
  4. 준공내역
    가. 수차발전기
      1) 위    치 : 경상남도 진주시 내동면 삼계리 1088
      2) 면    적 : 1,068㎡
      3) 수차형식 : 횡축 Kaplan Bulb type
      4) 발전기 형식 : 동기발전기
      5) 시설용량 : 18,000㎾(9,000㎾×2대)
      6) 사용수량 : 113톤/초(2대 기준)
　
  5. 사업시행기간 : 2016년 ∼ 2021년
