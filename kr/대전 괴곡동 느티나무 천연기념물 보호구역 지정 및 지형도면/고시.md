# 대전 괴곡동 느티나무 천연기념물 보호구역 지정 및 지형도면

1. 고 시 명 : 「대전 괴곡동 느티나무」천연기념물 보호구역 지정 및 지형도면
　
2. 고시사항
 가. 대상문화재 : 천연기념물 제545호 「대전 괴곡동 느티나무」 
  ㅇ 소 재 지 : 대전광역시 서구 괴곡동 985 등
 나. 보호구역 지정 면적 : 12필지(4,687㎡)

</img>
 다. 보호구역 지정 후 면적
 ㅇ 문화재구역 : 389㎡(2필지)
 ㅇ 보호구역 : 4,687㎡(12필지)

‘대전 괴곡동 느티나무’ 보호구역 지정 지번별 면적조서
<img id="30465608">
</img>
 라. 보호구역 지정 사유
  ㅇ 대전 괴곡동 느티나무와 마을 진입로 등이 인접하여 나무뿌리 생육에 지장을 줄 우려가 있으므로 느티나무 생육환경을 개선하고 향후 방문객 편의 시설 등을 조성하고자 보호구역을 지정함
 마. 관리단체 : 대전광역시 서구청
 바. 지형도면 : 붙임과 같음
　
3. 보호구역 지정일 : 관보 고시일
　
4. 특기사항 : 토지이용규제기본법 제8조에 따른 지형도면 등은 문화재공간정보(GIS)서비스(http://www.gis­heritage.go.kr)에서 열람이 가능하고, 문화재청 천연기념물과와 대전광역시 서구청에서 출력된 도면을 열람하실 수 있습니다.
