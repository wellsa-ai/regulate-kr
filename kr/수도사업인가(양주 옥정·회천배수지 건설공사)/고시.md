# 수도사업인가(양주 옥정·회천배수지 건설공사)

1. 사업명 : 양주 옥정?회천배수지 건설공사
2. 사업시행자
  ○ 명칭 : 양주시장
  ○ 주소 : 경기도 양주시 남방동 1-1번지
3. 사업의 목적 및 개요
  ○ 목적
    - 양주신도시(옥정?회천)택지개발지구 계획에 따른 신규 급수수요와 지구 내 급수구역의 안정된 급수체계 구축
  ○ 개요
    - 수원 : 수도권광역상수도 5·6단계(의정부계통)
    - 배수지 시설용량 : 40,000㎥
    ○ 송수관로 : L = 1,009.52m(D1,000mm)
    ○ 배수본관 : L = 1,173.26m(D900~1,000mm)
4. 사업시행지의 위치
  ○ 경기도 양주시 고암동 일원
5. 급수지역, 급수인구, 급수량, 사업시행기간 및 급수시작예정일
  ○ 급수지역 : 양주시 옥정동, 율정동, 고암동, 회암동, 삼숭동, 회정동, 덕정동, 덕계동, 고암동, 산북동 일원
  ○ 급수인구 : 180,616인(목표년도 2015년)
  ○ 일 최대 급수량 : 75,137㎥/일
  ○ 사업시행기간 : 2009.09 ~ 2011.12
  ○ 급수시작예정일 : 2012.01
6. 수용·사용할 토지조서 : 별첨
7. 관련도서는 양주시에 비치하고 이해관계자에게 열람(관련도서 생략).  끝.
