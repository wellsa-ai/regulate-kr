│구  분      │범   위                                                                   │

│보통주자본  │ -보통주 발행으로 인한 자본금(이와 유사한 출자금을 포함한다) 및 자본      │

│            │잉여금                                                                    │

│            │ - 기타 자본잉여금(보통주 외 기타자본증권의 발행에 따른 자본잉여금은      │

│            │제외)                                                                     │

│            │ - 이익잉여금                                                             │

│            │ - 기타포괄손익누계액                                                     │

├──────┼─────────────────────────────────────┤

│기타기본자본│ - 기타기본자금 발행으로 인한 자본금 및 자본잉여금                        │

│            │ - 제26조제2항에 따라 감독원장이 정하는 기준을 충족하는 신종자본증권      │

│            │등 자본증권                                                               │

├──────┼─────────────────────────────────────┤

│보완자본    │ -보완자본 발행으로 인한 자본금 및 자본잉여금                             │

│            │ - 자산건전성 분류결과 “정상” 및 “요주의”로 분류된 자산에 대하여 적립 │

│            │된 대손충당금등                                                           │

│            │ - 제26조제2항에 따라 감독원장이 정하는 기준을 충족하는 후순위채권        │

│            │등 자본증권                                                               │

├──────┼─────────────────────────────────────┤

│공제항목    │ - 보통주 자본에서 공제                                                   │

│            │   ? 영업권 상당액, 기타 무형자산, 이연법인세자산, 확정급여형 연금자산,  │

│            │유동화거래 매각이익                                                       │

│            │ - 해당 자본에서 공제                                                     │

│            │   ? 주식할인발행차금, 자기주식 및 자기발행 자본증권 보유금액            │

│            │   ? 자기자본비율 제고를 목적으로 타 금융기관과 상호보유한 자본증권      │

│            │ - 은행의 손실에 충당할 수 없는 자산 또는 자본항목으로서 감독원장이       │

│            │정하는 사항                                                               │
