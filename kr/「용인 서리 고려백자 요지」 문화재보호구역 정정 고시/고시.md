# 「용인 서리 고려백자 요지」 문화재보호구역 정정 고시

1. 고 시 명 : 「용인 서리 고려백자 요지」 문화재보호구역 정정 고시
 가. 대상 문화재
  1) 종    별 : 사적 제329호 용인 서리 고려백자 요지(1989.01.14. 지정)
  2) 소 재 지 : 경기도 용인시 처인구 이동면 중덕로 7 외 일원
  3) 사적 지정 정정내용 : 지적측량(분할) 등에 따른 문화재보호구역 증가면적(73.0㎡) 정정
  ㅇ 기존 지정면적 : 18,906.4㎡(문화재구역 4,264.4㎡ / 보호구역 14,642㎡)
  ㅇ 정정 후 면적 : 18,979.4㎡(문화재구역 4,264.4㎡ / 보호구역 14,715㎡)
  4) 문화재보호구역 정정사유 : 국가지정문화재(사적) 문화재보호구역 지정에 따른 지적(분할)측량 결과 3필지 73.0㎡ 면적 변경에 대하여 정정 고시
　
2. 정정 고시일 : 관보 고시일
　
3. 특기사항 
 ㅇ 「토지이용규제 기본법」제8조(지역·지구등의 지정 등)에 따른 지형도면 등은 ‘문화재공간정보(GIS)서비스(http://www.gis-heritage.go.kr)알림마당-지형도면고시’에서 열람이 가능하고, ‘경기도 문화유산과’ 및 ‘문화재청 보존정책과’에 관계도서를 비치하여 이해관계인에게 보이고 있습니다.
