# 수도사업변경인가(학용정수장 고도정수처리시설 설치사업)

1. 사업명 : 학용정수장 고도정수처리시설 설치사업
　
2. 사업시행자
　
 □ 명 칭 : 여수시장
 □ 주 소 : 전라남도 여수시 시청로 1
　
3. 사업의 목적 및 개요
 □ 목적 : 기존 노후화된 정수장에 막여과 정수공정을 도입하여 안전하고 고품질의 수돗물을 공급하기 위함.
 □ 개 요
   ○ 수 원 : 주암 조절지댐
   ○ 시설용량(변경)
     - 당  초 : 42,200㎥/일
     - 변  경 : 42,000㎥/일
   ○ 시설내용
     - 정수시설(변경)
      ? 당  초 : 42,200㎥/일(급속여과 42,200㎥/일)
      ? 변  경 : 42,000㎥/일(막여과 42,000㎥/일)
　
4. 사업시행지의 위치 및 면적(변경없음)
 □ 위  치 : 여수시 여천길 10번지 일원
 □ 면  적 : 49,590㎡
　
5. 급수구역 : 학용정수장 권역(여천권, 소라면, 율촌면, 화양면 일원)
　
6. 급수인구, 급수량, 사업시행기간 및 급수개시예정일
 □ 급수인구 : 123,693명
 □ 평균급수량(ℓ/일) : 300
 □ 급수개시(예정)일 : 2019년 1월
 □ 사업기간 : 2014년 11월 ~ 2018년 12월
　
7. 관계도서는 여수시청에 비치하고 이해관계인에게 열람(관계도서생략)
