# 통계작성의 승인(협의) 고시(국내체류외국인이동통계)

1. 통계의 명칭 : 국내체류외국인이동통계
　
2. 통계작성기관의 명칭 : 법무부
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제111016호 [2026.04.07.]
　
4. 통계작성의 목적 : 국내체류외국인의 지역 간 이동 실태를 체계적으로 파악하여 지역별 외국인 수요를 고려한 외국인정책 수립과 지역기반 이민정책 추진의 통계적 근거를 마련
　
5. 통계작성의 대상
       - 대상 : 개인/개인
       - 모집단 : 표본조사가 아니라서 모집단이 없습니다.
       - 대상 범위 및 규모 : 기준시점 동안 국내 장기체류 중인 외국인 전체
       - 대상지역 : 전국
　
6. 통계작성의 주기 : 월
　
7. 통계작성의 방법 : 보고통계/일반통계
