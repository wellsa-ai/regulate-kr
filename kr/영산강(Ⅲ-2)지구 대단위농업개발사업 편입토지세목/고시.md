# 영산강(Ⅲ-2)지구 대단위농업개발사업 편입토지세목

1. 사 업 명 : 영산강(Ⅲ-2)지구 대단위농업개발사업
　
2. 사업목적 : 수자원 확보, 농경지 확장, 식량증산, 농가소득 증대
　
3. 사업구역 : 1도 1군 4읍면
             전라남도 해남군 산이면, 황산면, 문내면, 화원면
　
4. 사업시행면적 : 7,840ha
　
5. 사업시행자 : 한국농어촌공사 사장
　
6. 사업기간 : 1989년 ∼ 2021년(33년)
　
7. 수용 또는 사용할 토지 세목조서 : 별항
　
8. 기타사항
 ㅇ 기타 자세한 사항은 한국농어촌공사 영산강사업단(전남 목포시 영산로 463, 전화 061-270-6481)에 문의 바랍니다.
