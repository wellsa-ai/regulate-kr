# 통계작성의 승인(협의) 고시(수질TMS부착사업장배출량현황)

1. 통계의 명칭 : 수질TMS부착사업장배출량현황
                       (Emissions from workplaces with Water-TMS attached)
　
2. 통계작성기관의 명칭 : 한국환경공단
　
3. 통계작성승인(협의)번호 [작성승인(협의)일]: 제392006호 [2024.4.16.]
　
4. 통계작성의 목적 : 수질TMS 측정기기 부착사업장의 수질오염물질 배출량 현황 공개를 통해 수질분야 정책 수립 및 국민 기초자료 제공
　
5. 통계작성의 대상 : 전국 수질TMS 부착사업장
　
6. 통계작성의 주기 : 분기
　
7. 통계작성의 방법 : 보고통계/일반통계
