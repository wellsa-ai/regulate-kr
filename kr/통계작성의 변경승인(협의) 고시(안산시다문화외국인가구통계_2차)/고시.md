# 통계작성의 변경승인(협의) 고시(안산시다문화외국인가구통계_2차)

1. 통계의 명칭 : 안산시다문화외국인가구통계
　
2. 통계작성기관의 명칭 : 경기도 안산시
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제619004호 [2019.09.30.]
　
4. 통계작성의 변경 연월일: 2025.12.19.
　
5. 통계작성의 변경 내용과 사유:
  ㅇ (다문화가구) 인구동태 부문 주석 변경
     - 변경 전(2024.1.1.~12.31.) → 변경 후(2023.1.1.~12.31.)
    ㅇ (외국인ㆍ외국인가구) 자산 부문 주석 변경
     - 주석3) 삭제
