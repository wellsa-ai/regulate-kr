# 「울산 경상좌도병영성」국가지정문화재(사적) 보호구역 추가 지정

1. 고 시 명: 「울산 경상좌도병영성」국가지정문화재(사적) 보호구역 추가 지정
　
2. 고시사항
  가. 대상문화재: 사적 「울산 경상좌도병영성」
   ㅇ 소재지: 울산광역시 중구 서동, 동동, 남외동 일원
   ㅇ 관리단체: 울산광역시 중구청장
  나. 추가지정 면적: : 4필지 235㎡
    * 세부사항은 붙임의 지번별 면적조사 참조
  다. 추가 지정 사유
   ㅇ 문화재보호구역 추가지정 지역은 울산 경상좌도병영성의 서-남구 간 체성에 접한 곳으로 현재 잔존 성벽이 잘 남아있어 보호구역으로 추가 지정하여 보존할 필요가 있음.
　
3. 지 정 일: 관보 고시일
　
4. 특기사항 
  ㅇ「토지이용계획 기본법」제8조(지역ㆍ지구등의 지정 등)에 따른 지형도면 등은 ‘문화재공간정보(GIS)서비스(http://www.gis-heritage.go.kr) 알림마당-지형도면고시’에서 열람이 가능합니다.
