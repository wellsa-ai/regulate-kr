# 화옹지구(2, 3공구) 대단위농업개발사업 준공검사 확인

1. 면허일자 : 1991년 3월 30일(농림수산부 조성 27245-368)
　
 2. 사 업 명 : 화옹지구(2, 3공구) 대단위농업개발사업
　
 3. 준공검사 확인을 받은 자의 성명·주소·연락처
    가. 성  명 : 한국농어촌공사 사장 최규성
    나. 주  소 : 전라남도 나주시 그린로 20
    다. 연락처 : 061-338-5385
　
 4. 매립목적 : 농지조성 등 농어촌 발전기반 조성
　
 5. 공사위치 : 경기도 화성시 서신·남양·마도·장안·우정(1도 1시) 일원 공유수면
　
 6. 준공면적 : 총 2,371,928.4㎡
    가. 국가귀속(공공용지) : 2,371,928.4㎡
      (화옹지구 2공구 : 1,957,476.4㎡, 화옹지구 3공구 : 414,452.0㎡)
　
 7. 매립공사의 시행기간 : 2007년 4월 16일 ~ 2013년 6월 30일
　
 8. 총사업비 : 93,444,304천원
      (화옹지구 2공구 : 53,196,443천원, 화옹지구 3공구 : 40,247,861천원)
　
 9. 기타사항 : 기타 문의사항은 한국농어촌공사(061-338-5385)로 문의하시기 바랍니다.
