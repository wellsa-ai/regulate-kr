# 통계작성의 승인 고시(국가연구개발사업통계)

1. 통계의 명칭 : 국가연구개발사업통계
　
2. 통계작성기관의 명칭 : 미래창조과학부
　
3. 통계작성승인번호(작성승인일) : 제12703호(2016. 02. 01.)
　
4. 통계작성의 목적 : 국가연구개발사업의 집행내역을 파악하여 국가연구개발사업 추진 방향과 정책 수립을 위한 기초자료 제공
　
5. 통계작성의 대상 : 매년 정부예산(일반회계, 특별회계, 기금)으로 편성된 국가연구개발사업 전수
　
6. 통계작성의 주기  : 1년
　
7. 통계작성의 방법  : 보고통계
