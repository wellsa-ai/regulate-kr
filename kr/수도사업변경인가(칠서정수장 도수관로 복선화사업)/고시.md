# 수도사업변경인가(칠서정수장 도수관로 복선화사업)

[1] 일반수도사업인가 고시
 1. 사업명 : 칠서정수장 도수관로 복선화사업
 2. 사업시행자의 명칭 및 주소 : 창원시장(경상남도 창원시 의창구 중앙대로 151)
 3. 사업의 목적 및 개요 : 칠서정수장 계통의 수로관로 복선화 설치를 통해 원수 공급의 안정성을 도모
 4. 사업시행지의 위치 : 경상남도 함안군 칠서면 계내리 일원
 5. 사업시행기간 : 2017. 5. ~ 2019. 12.(기간변경) 
 6. 수용·사용할 토지 등의 조서(변경) : 붙임 내용과 같음

[3] 함안군계획시설(수도공급설비)사업 실시계획 인가 고시
 1. 사업시행지의 위치 : 함안군 칠서면 계내리 일원(칠서정수장↔칠서취수장)
 2. 사업의 종류 및 명칭
   - 종 류 : 함안군계획시설(수도공급설비) 사업
   - 명 칭 : 칠서정수장 도수관로 복선화사업
 3. 사업의 면적 또는 규모
   - 면 적 : 당초 9,550㎡ → 변경 9,536㎡(14㎡ 감소)
   - 규 모 : 0.855km
 4. 사업시행자의 성명 및 주소
   - 성 명 : 창원시장
   - 주 소 : 창원시 의창구 중앙대로 151(용호동)
 5. 사업의 착수 및 준공예정일
   - 착수예정일 : 실시계획 고시일
   - 준공예정일 : 2019년 12월 31일
 6. 수용 또는 사용할 토지 등의 세목조서(변경) : 붙임 내용과 같음
