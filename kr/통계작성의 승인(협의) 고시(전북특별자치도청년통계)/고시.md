# 통계작성의 승인(협의) 고시(전북특별자치도청년통계)

1. 통계의 명칭 : 전북특별자치도청년통계
　
2. 통계작성기관의 명칭 : 전북특별자치도
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제214011호 [2024.09.30.]
　
4. 통계작성의 목적 : 전북특별자치도 청년의 인구, 경제, 일자리, 창업, 복지, 건강 등에 대한 통계를 생산ㆍ분석하여 청년 정책 수립의 기초자료로 활용
　
5. 통계작성의 대상
       - 대상 : 개인/가구
       - 모집단 : 2022. 12. 31. 현재 전북특별자치도 인구 중 만 18~39세
       - 대상 범위 및 규모 : 전북특별자치도 청년(만18~39세) / 개인, 1,774,248명 기준
       - 대상지역 : 시도
　
6. 통계작성의 주기 : 2년
　
7. 통계작성의 방법 : 가공통계/일반통계
