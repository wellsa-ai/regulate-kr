# 통계작성의 승인(협의) 고시(대구광역시북구청년통계)

1. 통계의 명칭 : 대구광역시북구청년통계
　
2. 통계작성기관의 명칭 : 대구광역시 북구
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제555002호 [2024.12.02.]
　
4. 통계작성의 목적 : 생애주기 중 교육, 고용시장 진입 및 유지를 통해 개인의 경제ㆍ사회적 삶의 수준과 질을 결정하는 시기인 청년(19~39세)의 삶을 다차원적으로 생산ㆍ분석하여, 삶의 질 향상을 위한 청년정책 수립의 기초자료로 제공하고자 함
　
5. 통계작성의 대상
       - 대상 : 개인/개인
       - 모집단 : 기준시점 현재, 대구광역시 북구 주민등록인구 중 19~39세
       - 대상범위 : 기준시점 현재, 대구광역시 북구 주민등록인구 중 19~39세
       - 대상규모 : 대구광역시 북구 청년(19~39세 약 109,067명 / 개인
       - 대상지역 : 시군구
       - 작성항목 : 7개 부문 75개 항목
                        [인구(20), 경제(10), 일자리(13), 창업(5), 복지(8), 건강(8),  삶의 질(11) 등]
　
6. 통계작성의 주기 : 2년
　
7. 통계작성의 방법 : 가공통계/일반통계
