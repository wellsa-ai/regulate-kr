┌─────┬───────────────────────────────────────────────┐

│제목      │ A와 B의 업무 공동수행 실시 협약서                                                            │

├─────┤                                                                                              │

│전문      │A와 B는 업무의 공동수행을 위해 다음과 같이 협약을 체결한다.                                   │

├─────┤                                                                                              │

│총칙조항  │제1조 (목적) 본 협약서는 양 기관이 업무의 공동수행을 위해 필요한 기본적인 사항을 규정함을     │

│          │목적으로 한다.                                                                                │

│          │제2조 (기본 운영원칙) 양 기관은 업무를 공동수행함에 있어 당해 기관의 제 규정을 준수하고 상호  │

│          │호혜적인 교류 및 협력관계를 유지한다.                                                         │

├─────┤                                                                                              │

│실체조항  │제3조 (협력분야) 양 기관이 추진하는 공동수행 업무의 내용은 다음과 같으며 구체적인 사항은      │

│          │상호 협의하여 결정한다.                                                                       │

│          │      1.                                                                                      │

│          │      2.                                                                                      │

│          │      3.                                                                                      │

│          │제4조 (인력 활용 및 교류) 양 기관은 필요한 경우 상호 협의하여 연구요원 등의 활용 및 교류를    │

│          │할 수 있으며 방법은 양 기관의 규정에 따른다.                                                  │

│          │제5조 (학술정보 교류) 양 기관은 학술 및 기술정보의 원활한 교류를 통해 긴밀한 협조관계를 유지하│

│          │며 학술 및 기술정보의 공동 활용에 적극 노력한다.                                              │

│          │제6조 (자문) 양 기관은 상호 간 공동업무 수행에 필요한 자문에 성실히 응한다.                   │

│          │제7조 (기타) 기타 필요한 사항에 대해서는 양 기관이 상호 협의하여 결정한다.                    │

├─────┤                                                                                              │

│해석조항  │제8조 (해석) 본 협약서에 명시되지 않은 사항과 협약서의 조문 해석이 서로 다른 경우에는 상호 협 │

│          │의하여 결정한다.                                                                              │

├─────┤                                                                                              │

│효력 및   │제9조 (효력 및 유효기간) ①본 협약은 양 기관의 협약 당사자가 서명한 날부터 효력이 발생한다.   │

│ 해지 조항│  ②협약의 유효기간은 효력이 발생한 날부터 0년으로 하며, 유효기간 종료일에 자동 종료된다.     │

│          │  ③본 협약의 필요성이 상실되는 경우에는 상호 협의하여 조기 종료할 수 있으며, 연장이 필요한   │

│          │경우 상대측에 0개월 이전에 서면으로 통보하거나 상호 협의하여 연장할 수 있다.                  │

├─────┤                                                                                              │

│개정조항  │제10조 (개정) 본 협약서의 내용은 상호 협의에 의하여 개정할 수 있다.                           │

├─────┤                                                                                              │

│기타 부칙 │제11조 (비용부담) 공동업무를 위해 소요되는 비용은 상호 협의하여 결정한다.                     │

│          │제12조 (운영) 본 협약서의 내용을 효과적으로 운영하기 위하여 상호 협의 하에 연락담당자를 둔다. │
