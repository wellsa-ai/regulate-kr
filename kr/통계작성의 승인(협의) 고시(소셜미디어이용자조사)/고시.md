# 통계작성의 승인(협의) 고시(소셜미디어이용자조사)

1. 통계의 명칭 : 소셜미디어이용자조사 (Social Media Users in Korea)
　
2. 통계작성기관의 명칭 : 한국언론진흥재단
　
3. 통계작성승인(협의)번호 [작성승인(협의)일]: 제413003호 [2024.10.15.]
　
4. 통계작성의 목적 : 소셜미디어 이용자들의 이용 행태 등을 파악ㆍ분석하여 향후 미디어 환경변화에 대응할 수 있는 정책수립에 기초자료로 활용
　
5. 통계작성의 대상 : 가구 내 만 19세 이상 가구원
　
6. 통계작성의 주기 : 3년
　
7. 통계작성의 방법 : 조사통계/일반통계
