# 통계작성의 승인(협의) 고시(농지임차료실태조사)

1. 통계의 명칭 : 농지임차료실태조사
　
2. 통계작성기관의 명칭 : 농림축산식품부
　
3. 통계작성승인(협의)번호 [작성승인(협의)일] : 제114062호 [2025.10.01.]
　
4. 통계작성의 목적 : 신뢰성 있는 농지임차료 통계 생산을 통해 관련 정책의 기초자료를 제공하고, 농지임차료 정보제공을 통해 신규 진입농의 진입장벽 해소
　
5. 통계작성의 대상
       - 대상 : 기타/기타
       - 모집단 : 농업경영체DB에 등록된 임차 농지
       - 대상 범위 및 규모 : 전국 임차농지 (3,000 표본)
       - 대상지역 : 전국
　
6. 통계작성의 주기 : 1년
　
7. 통계작성의 방법 : 조사통계/일반통계
