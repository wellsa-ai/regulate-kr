# 국제우편 이용에 관한 수수료

1. 통상우편물
　가. 국제반신 우표권
　  ㅇ 판 매 : 1,450원
　  ㅇ 교 환 : 항공서장 4지역 20g 해당요금
　
　나. 등기료 : 4,270원
　
　다. 통관절차대행수수료
　  ㅇ 통관 대상 발송 우편물 : 1,000원
　  ㅇ 관세 부과된 도착 우편물 : 3,000원
　    (전자상거래 통관 우편물은 1,000원)
　  ㅇ 우편자루 배달 인쇄물 : 4,000원
　
　라. 행방조사 청구료
　  ㅇ 항공우편청구 : 무료
　  ㅇ 국제특급우편(EMS) 청구 : 해당요금
　
　마. 배달통지청구료(등기한) : 1,500원
　
　바. 주소변경 및 환부 청구료
　  ㅇ 외국으로 발송준비 완료 전
　    - 접수국 발송준비 완료 전 : 무료
　    - 접수국 발송 후 : 국내등기취급수수료
　  ㅇ 외국으로 발송준비 완료 후
　    - 항공우편 청구 : 1,800원
　    - 팩스 청구 : 4,800원
　
　사. 등기우편물 반송료(반송취급료) : 국내우편등기료(무료등기는 제외)
　
　아. 보험료
　  ㅇ 기본요금 : 550원
　  ㅇ 추가배달료(보험가입 시 필수) : 1,300원
　  ㅇ 추가요금(보험가액 65.34 SDR 초과마다) : 550원
　
　자. 국제우편요금 수취인 부담(IBRS) 취급 수수료
　  ㅇ 인쇄물(봉투) 50g까지 : 1,100원
　  ㅇ 엽  서 : 500원
　
2. 특급우편물(EMS)
　가. 통관절차대행수수료
　  ㅇ 관세 부과된 도착 우편물 : 4,000원
　     (전자상거래 통관 우편물은 1,000원)
　
　나. 배달통지청구료 : 1,500원
　
　다. 주소변경 및 환부 청구료
　  ㅇ 외국으로 발송준비 완료 전
　    - 접수국 발송준비 완료 전 : 무료
　    - 접수국 발송 후 : 국내등기취급수수료
　  ㅇ 외국으로 발송 후
　    - 항공우편 청구 : 1,800원
　    - 팩스 청구 : 4,800원
　
　라. 초특급 서비스 수수료
　  ㅇ 접수익일(J+1) : 4,500원
　  ㅇ 대상국가(도시) : 홍콩, 베트남(하노이, 호치민)  
　  ㅇ 접수관서 및 접수마감시각 : 서울ㆍ경인지방우정청 국내특급우편 취급 고시사항의 당일특급접수우체국 및 취급시간 참조
　
　마. 보험료
　  ㅇ 기본요금 : 2,800원
　  ㅇ 추가요금(보험가액 65.34 SDR 초과마다) : 550원
　
　바. EMS 방문접수 수수료(계약고객 제외)
　  ㅇ 1회 방문 1통 당 3,000원, 추가 1통 당 1,000원(최대 5,000원)
　
　사. 해외 전자상거래용 반품서비스(IBRS) 수수료
　  ㅇ 적용 대상 : 2kg 이하의 소형 물품 
　  ㅇ 취급 지역 : 일본
　  ㅇ 취급수수료(IBRS EMS) : 통당 10,000원
　
3. 소포우편물
　가. 통관절차대행수수료
　  ㅇ 관세 부과된 도착 및 반착 우편물 : 4,000원
      (전자상거래 통관 도착 우편물은 1,000원)
　
　나. 행방조사 청구료
　  ㅇ 항공우편청구 : 무료
　  ㅇ 특급우편(EMS) 청구 : 해당요금
　
　다. 배달통지청구료 : 1,500원
　
　라. 주소변경 및 환부 청구료 
　  ㅇ 외국으로 발송준비 완료 전
　    - 접수국 발송준비 완료 : 무료
　    - 접수국 발송 후 : 국내등기취급수수료
　  ㅇ 외국으로 발송 후
　    - 항공우편 청구 : 1,800원
　    - 팩스 청구 : 4,800원
　
　마. 등기우편물 반송료(반송취급료) : 배달국가의 반송요금
　
　바. 보험료
　  ㅇ 기본요금 : 2,800원
　  ㅇ 추가요금(보험가액 65.34 SDR 초과마다) : 550원
　
4. 특별인출권 환율(SDR)
　  ㅇ 우정사업본부장이 공고하는 바에 따름
　
5. 국제우편물 추가운송수수료 : 다음 각 목의 국제우편물에 대하여 우정사업본부장이 공고하는 바에 따름
　가. 국제항공우편물
　나. 국제선편우편물
