# Regulate KR

대한민국 금융 감독규정을 Git 저장소로 관리합니다. 각 규정은 Markdown 파일이고, 각 개정은 Git commit으로 추적됩니다.

[legalize-kr](https://github.com/legalize-kr/legalize-kr) (법률 Git 프로젝트)에서 영감을 받았습니다.

## 빠른 시작

```bash
git clone https://github.com/illussna/regulate-kr.git
cd regulate-kr

# 금융소비자보호 감독규정 보기
cat kr/금융소비자보호에관한감독규정/고시.md

# 특정 단어 검색
grep -r "적합성" kr/

# 은행업 감독규정과 저축은행 감독규정 비교
diff kr/은행업감독규정/고시.md kr/상호저축은행업감독규정/고시.md
```

## 구조

```
kr/{규정명}/
  고시.md          # 금융위원회 고시 (감독규정)
```

## legalize-kr과의 관계

- **legalize-kr**: 법률 · 시행령 · 시행규칙 (국회/대통령/부처 제정)
- **regulate-kr**: 감독규정 · 고시 · 행정규칙 (금융위원회/금감원 제정)

법률이 뼈대라면, 감독규정은 실무에서 실제로 적용되는 세부 규칙입니다. 금융회사 준법감시에는 둘 다 필요합니다.

## 데이터 출처

모든 규정 데이터는 [국가법령정보센터](https://www.law.go.kr)에서 가져옵니다. 행정규칙 원문은 대한민국 정부 공공저작물로 자유롭게 이용 가능합니다.
